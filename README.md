Google Play Store App Analysis
Assignment Requirement

To proceed with this assignment, students must have successfully completed Data Specialization Module (D2).
This is a mandatory assignment designed to strengthen practical data analysis skills.

Objective

The objective of this project is to analyze Google Play Store app data to uncover insights about app performance, user preferences, and factors influencing ratings, downloads, and revenue.

Tools Used
Python
Data Cleaning
Exploratory Data Analysis (EDA)
Data Visualization
Statistical Analysis
Files Included
google_play_store.csv → Raw dataset
analysis_notebook.ipynb → Data analysis and visualizations
app_insights.png → Key insights visualization (optional)
Key Insights
Free apps tend to have higher download counts than paid apps
Apps with higher reviews generally receive better ratings
Certain categories (like Games and Tools) dominate the market
Frequent updates positively impact user ratings
 Dashboard / Visualization Preview

Conclusion

This project demonstrates how data analysis techniques can be applied to real-world app data to generate actionable insights, helping businesses improve app quality and user engagement.
